-------------------------------------------------------------------------------------

-------------------------------------------------------------------------------------
| Ctrl + Shift+y                   | changes the editor text to lower case          |

| Ctrl + space                     | autocomplete                                   |

| Ctrl + H                         | to find/replace in file, project, or directory |
-------------------------------------------------------------------------------------
| Ctrl + L                         | go to line number ..                           |

| Ctrl + d                         | delete line                                    |

| Ctrl + q                         | go to the last edited place                    |
